# currency

A small currency conversion command.

# Setup

```shell
$ go build -o currency main.go
$ ./currency 50 GBP USD
50 GBP to USD is 66.99 USD
```

# Environment Variables

| Name | Description |
|----- | ----------- |
| `CURRENCYCONVERTER_FREECURRENCYAPI_KEY` | https://app.freecurrencyapi.com |
